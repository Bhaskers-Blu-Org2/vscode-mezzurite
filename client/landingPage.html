Mezzurite Extension

Overview

This plugin improves the Mezzurite SDK implementation experience. The Mezzurite SDK provides a consistent way to collect performance metrics of Single Page Applications across a variety of frameworks.
Learn more about Mezzurite by visiting our GitHub Repository!

Features provided by this extension

  • Components Tracking

    Displaying the marked and unmarked components for performance tracking from the application code currently active in editor workspace.

  • Modules Tracking

    Displaying the modules configured with mezzurite performance markers to capture ALT and VLT for the application code currently active in editor workspace.

  • Intellisense and autocomplete

    Suggestion are provided for mezzurite syntaxes while typing imports and other mezzurite related code. Code completion would be performed automatically while hitting Ctrl+Space on specific syntaxes.

  • Verifying Loggers

    This feature verifies whether logging options for captured metrics is enabled or not. This also provides suggestions on required mandatory properties in logging config.

Shortcuts

  • Validate mezzurite configuration with a shortcut command:- ctrl + alt + m or through command pallete with ctrl + shift + p and type "Validate Mezzurite Configuration"
  • Get mezzurite extension's guide through command pallete with ctrl + shift + p and type "Mezzurite Extension Guide"

Frameworks supported

  • AngularJs
  • Angular 2-5
  • Angular 6
  • React

Metric timing values

Mezzurite vs code extension does not provides details about app tracking, component tracking and timing values for ALT, VLT and CLT. For more details on these metrics and their values, please checkout our Chrome extension here.
